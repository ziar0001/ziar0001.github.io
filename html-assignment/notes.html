List of 5 html tags

This is a list of 5 html tags we learned

sup
it's an HTML tag that means superscript — it displays text a little higher and smaller than normal text.
p
It's an HTML tag that means paragraph.It's used to display a block of text as a paragraph on a webpage.
li
It's an HTML tag that means list item.It's used inside a list to represent each item in that list.
ol
is an HTML tag that means ordered list — a list where items are shown in a specific order (usually numbers)
dt
It's used inside a definition list to show the term (word or title) being defined.
